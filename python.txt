Переменная - именованная область памяти, предназначенная для хранения значения. 

Типы объектов:
Целое число - int (от слова integer)
Число с плавающей точкой - float
Строка - string
Логический тип - boolean
Список - list
Словарь - dict
Кортеж - tuple
Множество - set
Файлы - file


Операции над числами:
Сложение +
Вычитание -
Умножение *
Деление /
Возведение в степень **
Целочесленное деление //
Остаток от деления %

## Деление на цело всегда показывает целое число, 11 // 5 = 2 (показывает, сколько раз умещается 5-ка)

int + float = float
Деление всегда предоставляет значение Float
Вычисть из int - float = float
Умножение всегда дает float
Возведение в степень:
- int ** int = int
- float ** int = float (и наоборот)

Деление на цело:
12 // 2 = 6 (int)
8 // 10 = 0 (int)

Остаток от деления:
19 % 7 = 5 (int)

Приоритеты:
1) В скобках 2*(5+2) = 14
2) Возведение в степень 3*2**3 = 24
3) * / // %
4) + -

Встроенные операции:
1) abs(-7) -> 7 (модуль отбрасывает знак)
2) min(-10, 2, 5) -> -10
3) max (10, 2, -5) -> 10
4) pow (2, 3) -> 8, принимает на вход 2 значения, возводит первое число, в степень второго
5) round (2.6) -> 3, принимает на вход значение и выполняет округление по умолчанию до целого числа. 
Если указать через запятую значение, то округлит до данного значения после точки
6) Функция type(4.3) - покажет тип объекта
(class 'float')
