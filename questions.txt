Характеристики микросервисов?
- разбито на компоненты (контейнер продукта, баз данных, api и тд) Тем самым можно обновлять отдельно микросервисы, а не сразу весь монолитный кусок продукта

- децентрализованность
- stateless. Микросервис не должен сохранять на себе какие-либо данные, как минимум есть для этого базы данных
